Araucaria
![Araucaria](logo.png)

Aracauria is a command-line micro-games written in C++.

Requirements

    C++ compiler compatible with the C++23 standard
    Make

Compilation

    Clone the repository:

shell

git clone https://github.com/LordGriffo/Amazonia.git

    Navigate to the project directory:

shell

cd Araucaria

    Compile the project by running the make command:

shell

make

    The executables will be generated in the bin directory.
